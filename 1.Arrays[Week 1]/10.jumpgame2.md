GREEDY Solution
```java
class Solution {
    public int jump(int[] nums) {
        int jump=0;
        int currentEnd=0;
        int farthest=0;
        for(int i=0;i<nums.length-1;i++){
//each time while scanning through range we update the fartest index we can reach
            farthest=Math.max(farthest,i+nums[i]);
//when we reach end of current jump we must increment jump
            if(i==currentEnd){
                jump++;
                currentEnd=farthest;
            }
        }

        return jump;
    }
}
```
Complexity
Time: O(n)
Space: O(1)
Interview Explanation (30 seconds)

We treat each jump as covering a range of indices. While scanning all indices in the current range, 
we keep track of the farthest position that can be reached. When we reach the end of the current range, 
we must make a jump, so we increment the jump count and extend the range to the farthest reachable index.
This guarantees the minimum number of jumps because we always maximize the reach of each jump without making premature decisions.
