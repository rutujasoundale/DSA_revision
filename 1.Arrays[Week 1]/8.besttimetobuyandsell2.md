```java
class Solution {
    public int maxProfit(int[] prices) {
      int profit=0;
      for(int i=1;i<prices.length;i++){
        if(prices[i]>prices[i-1]){
            profit += prices[i] - prices[i-1];
        }
      } 
      return profit;
    }
}
```
Interview Explanation (30 seconds)

Since unlimited transactions are allowed, I don't need to search for one maximum profit. Every increasing price difference can be treated as an independent profitable transaction. If today's price is greater than yesterday's, I add that difference to the answer. This works because the sum of all positive increases over an increasing sequence equals the profit from buying at the beginning of the sequence and selling at the end. Therefore, a single pass through the array is enough.

Complexity
Time: O(n) (one traversal)
Space: O(1) (constant extra space)

This is a classic Greedy algorithm: at each step, you make the locally optimal choice (take every profitable increase), and those local choices add up to the globally optimal profit.
