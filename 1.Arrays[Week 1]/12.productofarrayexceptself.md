```java
class Solution {
    public int[] productExceptSelf(int[] nums) {
        int n=nums.length;
        int[] ans=new int[nums.length];
          for(int i=0;i<nums.length;i++){
            int product=1;
            for(int j=0;j<n;j++){
                if(i!=j){
                    product*=nums[j];
                }
            }
            ans[i]=product;

          }  
          return ans;
    }
}
```
Time Complexity (TC)
The outer loop runs n times.
For each iteration of the outer loop, the inner loop also runs n times.

So the total operations are:

n×n=n
2

TC = O(n²)

Space Complexity (SC)
The ans array stores n elements.
Apart from that, only one variable (product) is used.

If we count the output array (as interviewers usually do):

SC = O(n)

If we don't count the output array (some interviewers don't, since it's required by the problem):

Auxiliary Space = O(1)
