```java
class Solution {
    public int[] productExceptSelf(int[] nums) {
        int n=nums.length;
        int[] ans=new int[nums.length];
          for(int i=0;i<nums.length;i++){
            int product=1;
            for(int j=0;j<n;j++){
                if(i!=j){
                    product*=nums[j];
                }
            }
            ans[i]=product;

          }  
          return ans;
    }
}
```
Time Complexity (TC)
The outer loop runs n times.
For each iteration of the outer loop, the inner loop also runs n times.

So the total operations are:

n×n=n
2

TC = O(n²)

Space Complexity (SC)
The ans array stores n elements.
Apart from that, only one variable (product) is used.

If we count the output array (as interviewers usually do):

SC = O(n)

If we don't count the output array (some interviewers don't, since it's required by the problem):

Auxiliary Space = O(1)

###What if we need to optimize 
Think if we can get product from left side or right and when we multiply weget the ans
This is it 
Prefix product + suffix product == ans

```java
class Solution {
    public int[] productExceptSelf(int[] nums) {
        int n=nums.length;
        int[] left=new int[n];
        int[] right=new int[n];
        left[0]=1;
        for(int i=1;i<n;i++){
            left[i]=left[i-1]*nums[i-1];

        }
        right[n-1]=1;
        for(int i=n-2;i>=0;i--){
            right[i]=right[i+1]*nums[i+1];
        }
        int[] ans=new int[n];
        for(int i=0;i<n;i++){
            ans[i]=left[i]*right[i];
        }
        return ans;

    }
}
```
Complexity
Time: O(n) (three linear traversals)
Space: O(n) (for left and right arrays, excluding ans)


###Further optimized To reduce space to O(1)
1.Store left products directly in ans 
2.traverse from right to left (instead of creating right array)
3.stre current element in ans and multiply with suffix
4.then multiply suffix with  stored prefix products

```java
class Solution {
    public int[] productExceptSelf(int[] nums) {
        int n=nums.length;
        
        int[] ans=new int[n];
        ans[0]=1;
        for(int i=1;i<n;i++){
            ans[i]=ans[i-1]*nums[i-1];

        }
        int suffix=1;
        for(int i=n-1;i>=0;i--){
            ans[i]*=suffix;
            suffix=suffix*nums[i];
        }
        return ans;

    }
}
```
Complexity
Time: O(n)
Extra Space: O(1) (excluding the output array)

Interview one-liner:

First store the prefix (left) products in the answer array. Then traverse from right to left using a single suffix variable and multiply it with the stored prefix products to get the final answer.
