1.Brute force approach
Use splitting
Check every word keep updating answer with new length therefore the last non empty word encountered will be answer
Algorithm
1.Split the string by spaces.
2.Traverse the resulting array.
3.Ignore empty strings ("") because multiple spaces create empty elements.
4.Update length with the current word's length.
5.Return length.
```java
class Solution {
    public int lengthOfLastWord(String s) {
        String[] words=s.split(" ");
        int length=0;
        for(String word:words){
            if(!word.equals("")){
                length=word.length();
            }
        }
        return length;
    }
}
```
Time Complexity
O(n) (splitting + traversing)
Space Complexity
O(n) (creates an array of substrings)
Interview Note: Although this is often called the "brute force" solution because it uses extra memory and library functions, the preferred interview solution is to scan from the end of the string directly, achieving O(n) time and O(1) extra space.


2.Two pointer right to left approach
Think of it like what if we try to traverse array from end 
insted of checking every word lets only check and count length of last word which will reduce space however time complexity will be same as O(n)
```java
class Solution {
    public int lengthOfLastWord(String s) {
        int i=s.length() - 1;
        while(i>=0 && s.charAt(i)==' '){
            i--;
        }
        int count=0;
        while(i>=0 && s.charAt(i)!=' '){
            count++;
            i--;

        }
        return count;
    }
}
```
Complexity
Time Complexity: O(n) (at most one pass from the end)
Space Complexity: O(1)
Dry Run
s = "   fly me   to   the moon  "
Step 1: Skip trailing spaces
                          ↑
"   fly me   to   the moon  "

Move left until:

                        ↑
"   fly me   to   the moon"
Step 2: Count characters
n → count = 1
o → count = 2
o → count = 3
m → count = 4

Next character is a space, so stop.

Answer:

4


Intuition

Start from the end of the string.

Skip all trailing spaces.
Count characters until you encounter a space.
Return the count.

This avoids creating extra strings or arrays.

Algorithm
Set i = s.length() - 1.
While i >= 0 and s.charAt(i) == ' ', decrement i.
Initialize count = 0.
While i >= 0 and s.charAt(i) != ' ':
Increment count
Decrement i
Return count.
